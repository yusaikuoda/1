# Ledger Field Notes

Independent long-form stock research site.

Published files are static HTML/CSS and can be served from GitHub Pages or any static host.

Editorial boundaries:

- No paid coverage from covered companies.
- No buy/sell recommendations or price targets.
- No penny-stock promotion.
- Ads must remain separate from editorial content.

If using GitHub Pages, enable Pages from the repository settings and serve from the `main` branch root.
